# Images pour l'application

Ce dossier contient les images de style collaboratif pour l'application.

## Images à ajouter :

1. **collab.svg** - Image pour la page de connexion (illustration de collaboration)
2. **chat-empty.svg** - Image pour l'état vide (quand aucune conversation n'est sélectionnée)

## Suggestions d'images :

Vous pouvez utiliser des images de sites comme :
- https://undraw.co/ (illustrations SVG gratuites)
- https://storyset.com/ (illustrations animées et statiques)
- https://www.manypixels.co/gallery (illustrations de style moderne)

### Recommandations :

**Pour collab.svg** : 
- Cherchez des illustrations avec des personnes qui collaborent
- Thèmes : "teamwork", "messaging", "communication", "chat"
- Couleurs : emerald/teal pour correspondre au thème de l'app

**Pour chat-empty.svg** :
- Cherchez des illustrations d'état vide ou de messages
- Thèmes : "empty state", "no messages", "waiting", "chat bubble"
- Style minimaliste et épuré
